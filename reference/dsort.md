# Sort data frame

Sort data according to columns in data frame

## Usage

``` r
dsort(data, x, ..., decreasing = FALSE, return.order = FALSE)
```

## Arguments

- data:

  Data frame

- x:

  variable to order by

- ...:

  additional variables to order by

- decreasing:

  sort order (vector of length x)

- return.order:

  return order

## Value

data.frame
