# Set global options for `mets`

Extract and set global parameters of `mets`.

## Usage

``` r
mets.options(...)
```

## Arguments

- ...:

  Arguments

## Value

`list` of parameters

## Details

- `regex`: If TRUE character vectors will be interpreted as regular
  expressions (`dby`, `dcut`, ...)

- `silent`: Set to `FALSE` to disable various output messages

## Examples

``` r
if (FALSE) { # \dontrun{
mets.options(regex=TRUE)
} # }
```
