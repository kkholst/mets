# Simple linear spline

Simple linear spline

## Usage

``` r
LinSpline(x, knots, num = TRUE, name = "Spline")
```

## Arguments

- x:

  variable to make into spline

- knots:

  cut points

- num:

  to give names x1 x2 and so forth

- name:

  name of spline expansion name.1 name.2 and so forth
